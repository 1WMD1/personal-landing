# 🌳 朋友圈式时间线博客设计方案

## 📋 设计目标

打造一个**类似微信朋友圈**的博客展示方式：
- ✅ 按时间线排列，从上到下
- ✅ 像根须一样自然延伸
- ✅ 沉浸式浏览体验
- ✅ 移动端优先

---

## 🎯 核心设计概念

### "根须时间线" (Root Timeline)

```
      🌱 博客根节点
       |
    ┌──┴──┐
    |     |
  📝文章  📝文章
    |     |
    |   ┌─┴─┐
    |   |   |
  📝文章 📝文章 📝文章
    |
    |
  📝文章 (最新)
```

**视觉隐喻：**
- 时间像树根一样向下生长
- 每篇文章是根须上的一个节点
- 根须有分支（分类标签）
- 越往下越新（时间倒序）

## 🎨 视觉设计细节

### 1. 时间线样式

```css
/* 根须效果 */
.timeline {
  position: relative;
  padding-left: 40px;
}

/* 主根须 */
.timeline::before {
  content: '';
  position: absolute;
  left: 20px;
  top: 0;
  bottom: 0;
  width: 3px;
  background: linear-gradient(
    to bottom,
    #10b981 0%,    /* 翠绿 */
    #059669 30%,
    #047857 60%,
    #065f46 100%   /* 深绿 */
  );
  border-radius: 2px;
}

/* 节点 */
.timeline-node {
  position: relative;
  margin-bottom: 2rem;
}

/* 根须节点圆点 */
.timeline-node::before {
  content: '';
  position: absolute;
  left: -28px;
  top: 20px;
  width: 14px;
  height: 14px;
  background: #10b981;
  border: 3px solid white;
  border-radius: 50%;
  box-shadow: 0 0 0 3px #10b981;
  animation: pulse 2s infinite;
}

/* 根须分支效果 */
.timeline-node.branch::after {
  content: '';
  position: absolute;
  left: -20px;
  top: 27px;
  width: 20px;
  height: 2px;
  background: #10b981;
  transform: rotate(-30deg);
}
```

### 2. 动画效果

```css
/* 节点出现动画 */
@keyframes growDown {
  from {
    opacity: 0;
    transform: translateY(-20px) scale(0.95);
  }
  to {
    opacity: 1;
    transform: translateY(0) scale(1);
  }
}

.timeline-node {
  animation: growDown 0.6s ease forwards;
}

/* 脉冲效果 */
@keyframes pulse {
  0%, 100% {
    box-shadow: 0 0 0 3px rgba(16, 185, 129, 0.4);
  }
  50% {
    box-shadow: 0 0 0 6px rgba(16, 185, 129, 0.2);
  }
}
```

## 💡 创新点

### 1. "根须"视觉隐喻
- 时间线像树根向下生长
- 节点有脉冲动画（生命力）
- 颜色渐变（翠绿→深绿）

### 2. 混合内容类型
- 短内容（像说说）
- 图片流（像朋友圈）
- 长文章（技术分享）

### 3. 智能分类
根据标签自动选择卡片类型

### 4. 数据持久化
使用 localStorage 保存互动数据
